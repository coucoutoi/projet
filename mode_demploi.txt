Pour pouvoir utiliser et lancer votre sudoku solver, vous pouvez utiliser les 2 modes différentes:
	# LE MODE TEXTUEL #
		1ère méthode -> lancez un terminal et lancer le fichier main_projet.py suivi de la grille sous forme d'une suite de nombre avec des 0 pour symboliser une case vide (pour avoir plus d'informations sur les différentes options disponible de votre solver, lancer le fichier suivi de -h ou --help pour appliquer la fonction d'aide)
		2nde méthode -> ouvrez le fichier sudoku_solver.py dans un terminal idle puis lancez le programme, il vous suffira ensuite de créer une grille avec la fonction make_grid du module sudoku_grid (celui-ci est déjà importé, utiliser une ligne de code sous cette forme: >>> gird = sudoku_grid.make_grid(string) où string correspond à votre chaîne de caractère représentant la grille) et de lui apliquer la fonction search_sol (vous pourrez alors utiliser cette ligne de code: >>> search_sol(grid))

	# LE MODE GRAPHIQUE #
		1ère méthode -> lancez un terminal et lancer le fichier main_projet.py suivi seulement de l'option -gr (ou --graphical) puis vous pourez utiliser votre module graphique
		2nde méthode -> ouvrez le fichier graphical_solver.py dans un terminal idle puis lancez le programme, il vous suffira ensuite appeler la fonction main et vous pourrez utiliser votre solver
			Note: si vous voulez lancer le module graphique complet du programme, utilisez la fonction main() dans un terminal idle du fichier graphical_main.py
