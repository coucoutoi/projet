Pour utiliser lancer le solver, vous pouvez vous y prendre de 2 manières diféentes:
	- lancez un terminal et lancer le fichier main_solver.py suivi de la grille sous forme d'une suite de nombre avec des 0 pour symboliser une case vide (pour avoir plus d'informations sur les différentes options, lancer le fichier suivi de -h ou --help)
	- ouvrez le fichier sudoku_solver.py dans un terminal idle puis lancer le programme, il vous suffira ensuite de créer une grille avec la fonction make_grid du module sudoku_grid (celui-ci est déjà importé) et de lui apliquer la fonction search_sol
