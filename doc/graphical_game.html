graphical_game.create()[source]¶

This function creates the graphical board for a game. It also launches the event loop. Thus, this is the inly functino to run to have a functional graphical board.

graphical_game.help_popup()[source]¶

Creates a window to explain how do use the graphical game

graphical_game.__RAZ(button_grid)[source]¶

This funciton redraws the board with all cell’s values as the beginnig.

graphical_game.__clavier(button_grid, ind_line, ind_col, event)[source]¶

This function reads the event key system and attributes a value of the cell at coordonates ind_line,ind_col

graphical_game.__change(button_grid, ind_line, ind_col)[source]¶

this function attributes a new event for cells

graphical_game.__redraw(button_grid, ind_line, ind_col, solver=False)[source]¶

This function draws the board. Position ind_line and ind_col are used to test which number icon has to be drawn. It’s also manage it must to create a finished popup with solver.

graphical_game.__open_grid(button_grid, save=False)[source]¶

This function open a sudoku grid

graphical_game.__choose_diff(button_grid)[source]¶

This function manage a popup to choose a difficulty of grid.

graphical_game.__write_grid(button_grid, var, root)[source]¶

This function write a grid with th edifficulty choosen by the user.

graphical_game.__save()[source]¶

This function save your sudoku grid.

graphical_game.__solve(button_grid)[source]¶

This function resolved the sudoku grid and print one of its solutions

graphical_game.__correction(button_grid)[source]¶

This function manage the correction of values enter by the user.

graphical_game.__export()[source]¶

With this function, it’s possible to export the grid in the graphical solver module.

graphical_game.__back()[source]¶

This function destroy the game window and open the main window.
